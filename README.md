# Sadhana — MPSC Study Tracker 🎯

A premium, modern React web application built specifically for MPSC (Maharashtra Public Service Commission) aspirants to track their daily study routines, timer sessions, notes, and habits. 

## Features
- **Study Timer**: Pomodoro-style focus timer with session logging.
- **Daily Routine Schedule**: Plan out your day slot by slot.
- **Dashboard & Progress Analytics**: Visualize study streaks, heatmaps, and total study hours.
- **Subjects Management**: Track syllabus subjects with color coding.
- **Cloud Authentication**: Secure login and email verification powered by Supabase.
- **Habit Tracking**: Build positive daily habits like meditation, exercise, or reading.
- **Responsive Design**: Beautiful Material Design 3 interface, perfectly optimized for Desktop and Mobile viewing. 

## Technology Stack
- **Frontend**: Vite + React
- **Routing**: React Router DOM (v6)
- **Authentication & Database**: Supabase
- **Styling**: Pure Modern CSS (Material Design 3 Theme)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment
This project is built with Vite, making it extremely lightweight and fast. The recommended free hosting platform is [Vercel](https://vercel.com) or [Netlify](https://netlify.com). Simply connect this GitHub repository, add your Environment Variables, and click deploy!
